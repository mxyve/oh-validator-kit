# Validator 数据校验工具

鸿蒙ArkTS数据校验工具库，提供常用数据格式校验（手机号、邮箱、身份证号等）及自定义校验能力，适用于表单验证、用户输入检查等场景。

## 特性
- 覆盖常用校验场景：手机号、邮箱、身份证号、URL、密码强度等
- 支持自定义正则校验和多规则联合校验
- 零依赖，轻量高效
- 符合鸿蒙ArkTS开发规范，可直接打包为HAR包复用

## 安装
1. 将本模块作为Static Library引入项目（推荐），或打包为HAR包后依赖
2. 在需要使用的地方导入：
   ```typescript
   import { Validator } from '@hmos/validator-kit'; // 替换为实际包名或路径
